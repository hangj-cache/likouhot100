## 滑动窗口

```
// 右指针
for(int r = 0,l = 0; r < n; r++){
            char ch = arr[r]; // 当前考虑的元素（要加入的元素）
            while(l < r && set.contains(ch)){ // 左指针
                set.remove(arr[l]);
                l++;
                curLen--;
            }
            set.add(ch);
            curLen++;
            maxLen = Math.max(maxLen,curLen);
        }
```
